> A high-performance, terminal-themed portfolio interface built for security professionals and reverse engineers.

## // SYSTEM OVERVIEW
This application serves as a personal dossier and technical showcase, simulating a command-line interface (CLI) within a modern web browser. It features real-time GitHub API integration, hardware telemetry logging, and a secure contact transmission system.

### [ ARCHITECTURE ]
* **Core:** Node.js (Express framework)
* **Database:** SQLite3 (Local file-based storage)
* **Frontend:** Vanilla JavaScript + HTML5 + CSS3 (No framework bloat)
* **Security:** Helmet CSP, Rate Limiting
